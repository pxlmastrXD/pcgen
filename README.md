# PcGen: A very simple password generator in Rust.

## Installation
PcGen can be installed via cargo:
```shell
$ cargo install pcgen
```

Or, you can build it manually:
```shell
$ cargo build --release
$ cargo install --path .
```

## Use

```shell
$ pcgen 4
Output: 5346
```
